# wca-statistics
A collection of statistics regarding WCA competitions

## SQL
If you own a WCA[^1] account, you can visit the [statistics.worldcubeassociation.org](https://statistics.worldcubeassociation.org/) website and either compute your own queries or use one of mine from this repository. 
- Log into the WCA account
- Go to the *Database Query* section
- Copy and paste the code from the desired query
- submit!
